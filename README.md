# FastCat

FastCat 开源使用Tarui v2 + react + rust 开发的桌面Among Us模组管理器  

QQ群:[837094490](https://qm.qq.com/q/xgRhQ9pzhg)  

## Dev

### 预先准备

1.安装[NodeJs](https://nodejs.org/)  

2.安装 [Rust](https://www.rust-lang.org/tools/install)  

3.进入项目目录,打开控制台  

4.执行`pnpm i` 或 `pnpm install` 安装依赖  

### Web UI 开发

~~~ bash
# Web 调试
pnpm dev

# Web 构建
pnpm build
~~~

### 桌面开发

~~~ bash
# 桌面调试
pnpm tauri:dev

# 桌面构建
pnpm tauri:build
~~~
